# 打寶率對照表

打寶率就是你身上裝備的打寶率  
(傭兵打寶率 = 本尊打寶率 + 傭兵身上的打寶率)  
其他的分類就是變色成該物品品質`增加的機率`

物品掉落的原型是掉的瞬間決定  
打寶率不影響  
打寶率只影響他變色的機率  
所以符文掉落跟打寶率無關(因為他沒有變色)
