# 法術傷害詳解

## 法術傷害計算

總法術傷害(TD) = 法術基礎傷害(BD) \* 法術傷害(SD) \* 法術協同傷害(SF)

BD 就是技能本身的基礎傷害，跟`技能等級`還有`角色等級`有關，所以練等也很重要  
SD 就是裝備、護符還有技能加成的法術傷害總和
