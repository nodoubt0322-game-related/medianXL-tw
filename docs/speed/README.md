# 攻速器

打開官方的[攻速器](https://dev.median-xl.com/speedcalc/)網頁

Character 選你要算的職業  
Weapon Type 選武器種類  
Ｗ eapon Base 選武器原型  
因為是丟飛刀  
所以把 Throwing(丟躑) 也選起來

下面出現的表就是門檻了  
如圖，總共需要 60 攻速到達極速  
超過 60%的攻速都沒用  
同理，施法速度就是 163%  
點一下 Defensive Speeds  
就能看打擊恢復跟格擋恢復的門檻
