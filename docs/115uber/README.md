## 阿圖魯阿城

地獄難度 A2

就是一朵雲射一堆箭的  
防低、皮薄的很容易就被秒殺

右上角有一個史卡塔拉島  
有 120 跟 125 的任務在裡面

可以省點時間

## 薩卡蘭姆的耳朵

地獄難度 A4  
打死 A4 的依卒爾會出現一個紅門  
進去之後會先到鑽石之門  
再往上走會到銀光之城  
他有傳點要記得踩  
在往上走會到命運文庫  
就是圖中 Library of Fate 的位置

在命運文庫有六隻天使  
就是圖中的 Zakarum's Avatar  
天使本身是傷害免疫的  
你要引小怪到他旁邊  
殺死小怪會有一個 debuff 波  
天使受到影響會有短暫的時間不是傷害免疫

這邊有會放 punisher 的無敵砲塔  
被毒到瞬間血量變成 1  
記得喝水解毒回血  
天使有 1/6 的機率會掉護符
